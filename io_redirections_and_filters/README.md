**io_redirections_and_filters

This folder contains shell scripts to practice input/output redirections and text filtering commands. The files demonstrate how to redirect command output, manipulate files, and process text efficiently in Unix/Linux environments.

Files included
0-hello_world: Prints "Hello, World" to standard output.

1-confused_smiley: Displays a confused smiley face.

2-hellofile: Displays the contents of /etc/passwd.

3-twofiles: Displays contents of /etc/passwd and /etc/hosts.

4-lastlines: Displays the last 10 lines of /etc/passwd.

5-firstlines: Displays the first 10 lines of /etc/passwd.

6-third_line: Displays the third line of the file 'iacta'.

7-file: Creates a file named 'best_school' with 'Holberton School' inside.

8-cwd_state: Writes the result of ls -la to a file named 'ls_cwd_content'.

9-duplicate_last_line: Duplicates the last line of a file.

10-no_more_js: Deletes all files with the .js extension in the current directory and subfolders.

11-directories: Counts the number of directories and subdirectories in the current directory.

12-newest_files: Displays the 10 newest files in the current directory.

13-unique: Prints only lines that are unique in input.

14-findthatword: Displays lines containing the word 'root'.

15-countthatword: Displays the number of lines that contain the word 'bin'.

16-whatsnext: Displays the lines containing 'root' and the three lines following them.

17-hidethisword: Displays all lines that do not contain 'bin'.

18-letteronly: Displays all lines starting with a letter.

19-AZ: Displays all lines starting with a capital letter.

20-hiago: Removes all occurrences of 'c' and 'C' from input.

21-reverse: Reverses the contents of input.

22-users_and_homes: Displays all users and their home directories from the /etc/passwd file.

23-empty_casks: Displays all empty files in the current directory and subdirectories.

24-gifs: Lists all .gif files.

25-acrostic: Prints the first character of every line of input.

README.md: This file.

Usage
All files are shell scripts. Give execution permission before running:

bash
chmod +x <script_name>
Execute a script with:

bash
./<script_name>
Learning objectives
Master I/O redirections (>, >>, <, 2>, etc.).

Use standard filtering tools (like grep, sort, uniq, head, tail, wc, rev).

Automate file and text processing with shell scripts.

Work with permissions, directories, and various output formats.
